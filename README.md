# 🎮 اسم فامیل آنلاین | Esm-Famil Multiplayer Game

یک بازی کامل و رایگان اسم فامیل که هم به صورت **آفلاین** (چند نفره روی یک گوشی) و هم به صورت **آنلاین** (از راه دور روی گوشی‌های مختلف) اجرا می‌شه.

---

## 📂 ساختار پروژه

### 🔹 بازی (فرانت‌اند): ریپوی `esm-famil`
```
esm-famil/
├── index.html      ← فایل اصلی بازی
└── README.md       ← همین فایل
```

### 🔹 سرور (بک‌اند): ریپوی `esm-famil-server`
```
esm-famil-server/
├── index.js        ← کد سرور (Node.js + Socket.io)
└── package.json    ← وابستگی‌های سرور
```

> ⚠️ هر ریپو فقط فایل‌های خودش رو داره. فایل‌های اضافه رو حذف کن.

---

## ✨ ویژگی‌ها
- 📱 بازی آفلاین (نوبتی روی یک دستگاه)
- 🌐 بازی آنلاین (چند نفره از راه دور)
- 🔗 لینک دعوت + QR کد برای ورود سریع
- 🎲 دسته‌بندی پیش‌فرض + دسته‌بندی دلخواه
- ⏱️ تایمر + امتیازدهی خودکار (یکتا=۲۰، مشترک=۱۰)
- 📊 نتایج هر دور + نتایج نهایی 🏆

---

## 🚀 راهنمای دیپلوی

### ۱️⃣ سرور (`esm-famil-server`)
1. برو به [Render.com](https://render.com) → با GitHub لاگین کن
2. **New → Web Service** → ریپوی `esm-famil-server` رو انتخاب کن
3. تنظیمات:
   - **Build Command**: `npm install`
   - **Start Command**: `node index.js`
4. دکمه **Deploy** → صبر کن تا سبز بشه
5. آدرس سرور رو کپی کن (مثل: `https://esm-famil-server.onrender.com`)

### ۲️⃣ اتصال بازی به سرور
1. فایل `index.html` رو باز کن
2. خط ~۲۸۰ رو پیدا کن:
   ```javascript
   let SERVER_URL = 'https://esm-famil-server.onrender.com';
   ```
3. آدرس سرور خودت رو جایگزین کن

### ۳️⃣ انتشار بازی
1. برو به [Netlify Drop](https://app.netlify.com/drop)
2. فایل `index.html` رو بکش بنداز
3. لینک عمومی رو کپی کن → بازی آماده‌ست! 🎉

## 🎮 نحوه بازی

### 👑 میزبان
1. لینک بازی رو باز کن
2. اسمت رو بنویس → **🌐 آنلاین**
3. بازیکن‌ها و دسته‌بندی‌ها رو تنظیم کن
4. **ساخت اتاق** → لینک و QR رو بفرست
5. وقتی همه اومدن → **شروع بازی**

### 🙋‍♂️ بازیکن
1. روی لینک دعوت کلیک کن
2. اسمت رو بنویس → **ورود**
3. جواب‌ها رو بنویس → **ثبت جواب‌ها**
4. نتایج رو ببین و منتظر دور بعدی باش

## ❓ سوالات متداول

**آیا نیاز به نصب چیزی روی گوشی دارم؟**  
❌ نه! فقط کافیه لینک رو تو مرورگر باز کنی. `Add to Home Screen` بزن تا مثل اپ آیکونش بیاد.

**چرا بازیکن‌ها همدیگه رو نمی‌بینن؟**  
✅ آدرس سرور توی `index.html` دقیقاً همون آدرس Render باشه. سرور روی Render در حالت `Running` باشه.

**سرور رایگان Render بعد از مدتی خاموش می‌شه؟**  
✅ بله، بعد از ۱۵ دقیقه بی‌کاری Sleep می‌شه. بیدار شدنش ۳۰-۵۰ ثانیه طول می‌کشه.
